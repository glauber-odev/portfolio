BuzzFeed

O Angular BuzzFeed Game é uma aplicação web que utiliza o framework Angular, HTML, CSS e JavaScript. O objetivo é fornecer uma experiência divertida para os usuários. Consiste em uma série de perguntas que abordam aspectos da vida e personalidade do usuário. Com base nas respostas, o sistema calcula qual personagem do BuzzFeed mais se assemelha ao perfil do usuário.
